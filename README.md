# Blinkit Grocery Sales Data Analysis

## 📌 Project Overview

This project analyzes Blinkit's grocery sales data using **Python, SQL, Excel, and Power BI** to uncover business insights and create an interactive dashboard. The analysis focuses on sales performance, customer ratings, product categories, and outlet characteristics to support data-driven business decisions.

---

## 🎯 Objective

The objective of this project is to analyze Blinkit's sales data and answer key business questions that can help improve business performance.

The project focuses on:

* Understanding customer purchasing patterns
* Identifying high- and low-performing product categories
* Analyzing sales across different outlet types and locations
* Evaluating outlet performance using key business metrics
* Presenting insights through an interactive Power BI dashboard

The analysis answers the following business questions:

Key performance indicators include:

* Total Sales
